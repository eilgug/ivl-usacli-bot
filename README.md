# IVL US-ACLI BOT

## START PROJECT
- Start ngrok envirorment with `ngrok http <flask-port>`
- Copy ngrock host address in `.env` file
> Had `/` at the end of the url
- Run flask server `python <file-name>.py`
- Start bot calling `localhost:<port>`

## COMMANDS
- `/start`
- `/alive`
- `/classifica`
- `/nextmatch`
- `/calendario`
